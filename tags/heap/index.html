йўҳзӣ®е®Ўи®Ў

иҝҷйўҳејҖеұҖиҝҳжңү4з§’зҡ„sleepпјҢзңҹе®һжҒ¶еҝғпјҢжӢҝеҲ°shellеҗҺflagж–Үд»¶еҗҚиҝҳжҳҜдјҡеҸҳзҡ„

жң¬ең°и°ғиҜ•еҸҜд»ҘжҠҠsleepе’Ңalarm patchжҺү

еңЁaddеҠҹиғҪдёӯchunkзҡ„sizeдёҚз”ұжҲ‘д»¬жҺ§еҲ¶пјҢеӣәе®ҡ0x40

еңЁdeleteеҠҹиғҪдёӯеӯҳеңЁdouble freeзҡ„иЎҢдёәпјҢеҸӘиҰҒptrеӯҳеңЁпјҢеҚіеҸҜfreeпјҢеӣ жӯӨеҸҜд»Ҙй’ҲеҜ№дёҖдёӘchunkпјҢиҝһз»ӯfree

зЁӢеәҸжІЎжңүshowеҠҹиғҪпјҢеӣ жӯӨж— жі•зӣҙжҺҘиҺ·еҸ–ең°еқҖдҝЎжҒҜ

иҖғиҷ‘еҲ©з”Ёдҝ®ж”№IOFILEжқҘжі„жјҸдҝЎжҒҜпјҢдҪҶжҳҜиҝҷйҮҢйңҖиҰҒдёҖдёӘжҢҮеҗ‘libcзҡ„жҢҮй’ҲпјҢunsortedbinе°ұеҫҲеҗҲйҖӮпјҢдҪҶжҳҜиҝҷйҮҢchunkзҡ„sizeдёҚеҸ—жҺ§

дҪҶжҳҜжҲ‘д»¬еҸҜд»ҘеҲ©з”Ёdouble freeзҡ„ж•ҲжһңпјҢжқҘдҝ®ж”№nextжҢҮеҗ‘еҸҜд»ҘжҺ§еҲ¶жҹҗдёӘchunkзҡ„sizeзҡ„дҪҚзҪ®пјҢдҝ®ж”№е…¶дёәfastbinиҢғеӣҙд»ҘеӨ–зҡ„еҖјпјҢеӨҡж¬ЎfreeеЎ«ж»ЎtcahcheпјҢеҗҺйқўзҡ„е°ұдјҡиҝӣе…Ҙunsortedbin

жҺҘзқҖз”іиҜ·е°Ҹchunkе°ұеҸҜд»ҘеёҰеҮәlibcзҡ„ең°еқҖпјҢзҲҶз ҙ1/16пјҲеҸҜиғҪжҳҜеҗ§пјүжҢҮеҺ»stdout

еҗҢж ·еҲ©з”Ёdouble freeзҡ„ж•Ҳжһңж”№дёҖдёӘchunkзҡ„nextжҢҮеҗ‘еёҰзқҖlibcзҡ„chunkпјҢд»ҺиҖҢе°Ҷе…¶й“ҫдёҠtcache

з”іиҜ·еҲ°stdoutзҡ„ең°еқҖж—¶пјҢеҶҷе…Ҙ

жҺҘзқҖе°ұеҸҜд»Ҙжі„жјҸеҮәlibcең°еқҖпјҢеёёи§„зҡ„еҲ©з”Ёnextдҝ®ж”№free hookжӢҝshell

babypwn

иөӣеҗҺеҒҡеҮәжқҘзҡ„пјҢиҝҷйҮҢжІЎжңүејҖеҗҜPIE

йўҳзӣ®е®Ўи®Ў

иҝҷйўҳд№ҹеӯҳеңЁdouble freeзҡ„иЎҢдёә

еҗҢж ·д№ҹжІЎжңүshowеҠҹиғҪпјҢиҝҳжҳҜиҖғиҷ‘IOFILEжі„жјҸlibc

еҖјеҫ—жіЁж„Ҹзҡ„жҳҜиҝҷйҒ“йўҳдёҚдјҡfreeеҗҺдёҚжё…з©әchunk listпјҢеҸӘиғҪеӯҳж”ҫ10дёӘйқһ0еҖјпјҢеӣ жӯӨеҫ—жҺ§еҲ¶chunk list

chunk listеӨҙдёҠжңүдёүдёӘIOFILEзҡ„жҢҮй’ҲпјҢеҸҜд»ҘеҲ©з”ЁиҝҷйҮҢзҡ„0x7fжқҘеӯҳдёҖдёӘchunkпјҢд»ҺиҖҢжҺ§еҲ¶chunk list

е№¶дё”еңЁжң«е°ҫжҢүз…§йўҳзӣ®з»“жһ„еҹӢдёӢж–№дҫҝж—ҘеҗҺжё…з©әchunk listзҡ„жҢҮй’Ҳ

йҖҡиҝҮfreeиҝҷдёӘfake chunkдёҺmallocе°ұеҸҜд»ҘеҲ·ж–°chunk list

иҝҷж ·жңү7дёӘchunkжҳҜеҸ—ж”Ҝй…ҚпјҢдёҚиҝҮиҰҒдҝқз•ҷдёҖдёӘз”ЁжқҘеҲ·ж–°chunk list

жҺҘзқҖе°ұжҳҜеҲ©з”Ёdouble freeжқҘmallocеҺ»stdoutз»“жһ„дҪ“пјҢиҝҷйҮҢиҰҒеҲ©з”ЁstdoutдёҠйқўзҡ„stderrдёӯзҡ„дёҖдёӘй”ҷдҪҚжқҘmalloc

дҝ®ж”№stdoutдёә

жі„жјҸеҮәlibcеҗҺе°ұfastbin attackеҲ°malloc hookеҶҷone gadgetе°ұеҘҪдәҶ

дёҚиҝҮжҲ‘еңЁжң¬ең°иҜ•зҡ„ж—¶еҖҷпјҢone gadgetйғҪдёҚеҸҜз”ЁпјҢйғҪжңүдёҖдәӣйҷҗеҲ¶

з”ЁдәҶиҝҷдёӘone gdagetпјҢйңҖиҰҒrsp+0x30еӨ„дёә0пјҢи°ғиҜ•еҸ‘зҺ°иҝҷйҮҢдёҚдёә0пјҢдҪҶжҳҜд»–дёҠйқў3дёӘеҒҸз§»еӨ„жҳҜ0

з”ЁеҲ°reallocжқҘи°ғж•ҙж Ҳ

еңЁreallocдёӯдјҡжҳҜиҝҷж ·

жүҖд»ҘиҝҷйҮҢд»Һ0x846CCејҖе§Ӣжү§иЎҢпјҢpushдёүж¬ЎпјҢе°ұеҸҜд»Ҙж»Ўи¶і[rsp+0x30] == NULL

жүҖд»Ҙе°ұжҠҠmalloc_hookж”№еҲ°libcдёӯ0x846CCеӨ„пјҢеҶҚжҠҠrealloc_hookж”№дёәone gadgetпјҢеҚіеҸҜget shell
